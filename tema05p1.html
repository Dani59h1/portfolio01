REDESIGN

Once we had secured our client and had to start figuring out how to redesign his homepage, we began with some research. The first thing we did was a 5-second test to see what people thought of the current page after seeing it for only 5 seconds.

We then created a BERT test to see how it placed according to the desired result, a result we almost scored perfectly on in the end with our second BERT test.

Working in a group now, we introduced a scrum board. Something that would prove useful all the way through. It would always be there to remind us how far we’d come and what still remained to be done.

Working through Github is definitely not the same as working together on google drive. Every ‘push’ was with a little anxiety that something might go wrong, but, although not flawlessly, it still worked as intended.

Finally all I wish for is that we had more time if the project were to be finished.
